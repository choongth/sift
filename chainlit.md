# Sift — Personal Research Agent

Ask a research question and Sift will autonomously search the web, read sources, and produce a structured report.

**How it works:**
1. Sift proposes a research plan — you approve before it starts
2. It runs a multi-step search and synthesis loop
3. You approve the final report before it is saved
4. Report is saved as Markdown and uploaded to S3

**Example questions:**
- *What are the architectural differences between Kafka and RabbitMQ?*
- *Summarise the key findings from recent papers on RAG for long-context LLMs.*
- *What should I know before migrating a monolith to microservices?*
